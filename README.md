# Sayan Paul Portfolio

## Overview
This is a modern, responsive portfolio website for Sayan Paul, a Full Stack Developer & AI Enthusiast. The site showcases skills, projects, services, education, and contact information in a visually appealing and interactive manner.

## Features
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.
- **Smooth Animations:** Includes fade-in, parallax, and micro-interactions for engaging user experience.
- **Navigation:** Sticky navbar with active section highlighting and smooth scrolling.
- **Hero Section:** Personal introduction, stats, and social links.
- **About Section:** Background, tech stack, and expertise.
- **Skills Section:** Categorized skills with animated progress bars.
- **Services Section:** List of offered services with feature tags.
- **Projects Section:** Featured projects with live demo and code links.
- **Education & Certifications:** Timeline and certification cards.
- **Contact Section:** Contact info and interactive form with validation and toast notifications.
- **Download Resume:** Quick access to resume download.

## Technologies Used
- HTML5, CSS3, JavaScript (Vanilla)
- Font Awesome for icons
- Google Fonts for typography

## Setup & Usage
1. **Clone or Download:**
   - Clone the repository or download the source files to your local machine.
2. **Assets:**
   - Ensure the `assets` folder contains all images and the resume PDF.
3. **Open Locally:**
   - Open `index.html` in your browser to view the portfolio.
4. **Customization:**
   - Update content in `index.html` and styles in `styles.css` as needed.
   - Replace images and resume in the `assets` folder for personalization.

## File Structure
- `index.html` — Main HTML file
- `styles.css` — Custom styles
- `script.js` — Interactive JavaScript
- `assets/` — Images and resume

For feedback or collaboration, use the contact form on the site or reach out via email/LinkedIn.
